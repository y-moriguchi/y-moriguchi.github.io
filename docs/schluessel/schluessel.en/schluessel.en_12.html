3.6 Networing

Procedures for networking.

Function: open-tcp-input/output-port ip-address port buffer-mode transcoder

creates a input/output port using TCP.
buffer-mode and transcoder specifies the R6RS buffer mode and transcoder, respectively. If trancoder is #f, a port to be created will be a binary port.

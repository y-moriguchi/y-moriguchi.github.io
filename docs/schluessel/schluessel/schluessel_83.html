5.45 地理関連

地理関連の手続きです。 緯度オブジェクト・経度オブジェクトは加法、減法、スカラー倍が定義されています。

Function: make-latitude degrees
Function: make-latitude degrees minutes
Function: make-latitude degrees minutes seconds
Function: make-latitude degrees minutes seconds centiseconds

緯度オブジェクトを生成します。
degreesは0から90までの正確な数です。 minutesは0から60までの正確な数です。 secondsは0から60までの正確な数です。 centisecondsは0から100までの正確な数です。
正の緯度を北緯、負の緯度を南緯とします。南緯を入力するときは0でない最初の値を負の符号にします。

Function: make-longitude degrees
Function: make-longitude degrees minutes
Function: make-longitude degrees minutes seconds
Function: make-longitude degrees minutes seconds centiseconds

経度オブジェクトを生成します。
degreesは0から180までの正確な数です。 minutesは0から60までの正確な数です。 secondsは0から60までの正確な数です。 centisecondsは0から100までの正確な数です。
正の経度を東経、負の経度を西経とします。西経を入力するときは0でない最初の値を負の符号にします。

Function: string->latitude string

文字列より緯度を生成します。 

(string->latitude "10°10′10″N") -> 北緯10度10分10秒の緯度オブジェクト
(string->latitude "10°10′10″S") -> 南緯10度10分10秒の緯度オブジェクト
Function: string->longitude string

文字列より経度を生成します。 

(string->longitude "10°10′10″E") -> 東経10度10分10秒の経度オブジェクト
(string->longitude "10°10′10″W") -> 西経10度10分10秒の経度オブジェクト

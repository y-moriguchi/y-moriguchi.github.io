4.13 ベクタ

Function: vector? object

[R5RS] objectがベクタのとき#tを返します。

Function: vector object …

[R5RS] object…を要素に持つベクタを生成します。

Function: make-vector k [fill]

[R5RS] kこの要素を持つベクタを生成します。 ベクタの要素はfillで初期化されます。省略時は未定義です。

Function: vector-length vector

[R5RS] vectorの長さを取得します。

Function: vector-ref vector k

[R5RS] vectorからk番目の要素を取り出します。

Function: vector-set! vector k object

[R5RS] vectorからk番目の要素にobjectを設定します。

Function: vector-fill! vector fill [start end]

[R5RS] vectorの要素をすべてfillにします。 start(その場所を含む)、end(その場所を含まない)で 範囲を指定することもできます。

Function: vector->list vector

[R5RS] vectorをリストに変換します。

Function: list->vector list

[R5RS] 正しいリストlistをベクタに変換します。

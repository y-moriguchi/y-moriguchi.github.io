4.14 文字

Function: char? object

[R5RS] objectが文字のとき#tを返します。

Function: char<? char …
Function: char<=? char …
Function: char>? char …
Function: char>=? char …
Function: char=? char …

[R5RS] それぞれ左から右へ文字コードがより小さい、以下である、より大きい、 以上であり、等しいときに#tを返します。

Function: char-ci<? char …
Function: char-ci<=? char …
Function: char-ci>? char …
Function: char-ci>=? char …
Function: char-ci=? char …

それぞれ左から右へ文字コードがより小さい、以下である、より大きい、 以上であり、等しいときに#tを返します。
これらのメソッドは大文字小文字を区別しません。 [R5RS]

Function: char-alphabetic? char

[R5RS] 文字がアルファベットであるとき#tを返します。

Function: char-numeric? char

[R5RS] 文字が数値であるとき#tを返します。

Function: char-lower-case? char
Function: char-upper-case? char

[R5RS] 文字が大文字(小文字)のアルファベットであるとき#tを返します。

Function: char-whitespace? char

[R5RS] 空白文字のとき#tを返します。

Function: char-upcase char
Function: char-downcase char

[R5RS] 文字を大文字(小文字)に変換します。

Function: char->integer char
Function: integer->char char-code

[R5RS] 文字を文字コード、またはその逆に変換します。

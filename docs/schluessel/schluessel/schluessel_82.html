5.44.1 数詞

Function: number->english-numeral number

整数を英語で表した数詞に変換します。
変換できない数のときは#fを返します。

Function: english-numeral->number string

英語で表した数詞を整数に変換します。
無効な数詞のときは#fを返します。

Function: number->french-numeral number

整数をフランス語で表した数詞に変換します。
変換できない数のときは#fを返します。

Function: french-numeral->number string

フランス語で表した数詞を整数に変換します。
無効な数詞のときは#fを返します。

Function: number->german-numeral number

整数をドイツ語で表した数詞に変換します。
変換できない数のときは#fを返します。

Function: german-numeral->number string

ドイツ語で表した数詞を整数に変換します。
無効な数詞のときは#fを返します。

Function: number->roman-numeral number

整数をローマ数字に変換します。変換できる数は1から3999までです。
変換できない数のときは#fを返します。

Function: roman-numeral->number string

ローマ数字を整数に変換します。変換できる数は1から3999までです。
無効な数詞のときは#fを返します。

Function: number->big-roman-numeral number

整数を大きいローマ数字に変換します。 変換結果にUnicodeの文字を含むので環境によっては表示できない場合があります。
変換できない数のときは#fを返します。

Function: big-roman-numeral->number string

大きいローマ数字を整数に変換します。
無効な数詞のときは#fを返します。

Function: number->ionian-numeral number

整数をイオニア式ギリシャ数字に変換します。 変換結果にUnicodeの文字を含むので環境によっては表示できない場合があります。
変換できない数のときは#fを返します。

Function: ionian-numeral->number string

イオニア式ギリシャ数字を整数に変換します。
無効な数詞のときは#fを返します。

Function: number->latin-numeral number
Function: number->latin-numeral number case
Function: number->latin-numeral number case gender
Function: number->latin-numeral number case gender use-macron

整数をラテン語で表した数詞に変換します。
引数に格と性を指定できます。

nominative主格
genitive属格
dative与格
accusative対格
ablative奪格
muscline男性
feminine女性
neuter中性

マクロンを使用するときはuse-macronに#tを設定します。 変換できない数のときは#fを返します。

Function: latin-numeral->number string
Function: latin-numeral->number string use-macron

ラテン語で表した数詞を整数に変換します。 マクロンを認識するときはuse-macronに#tを設定します。
無効な数詞のときは#fを返します。

5.44.2 複数

Function: english-word->plural word

英単語を複数形に変換します。不規則変化に対してはある程度対応しています。 不規則変化でないと見なされると英語の規則に従って機械的に変換します。 従って、与えられた単語が英語の名詞であるかは判断しません。

Function: english-plural? word

与えられた英単語が複数形であると思われるときに#tを返します。 不規則変化に対してはある程度対応しています。 与えられた単語が英語の名詞であるかを判断しないので、 例えば動詞の3人称・単数・現在形に対しても#tを返すことがあります。

5.44.3 日本語ユーティリティ

Function: kana->romaji string
Function: kana->romaji string flags …

かなをローマ字に変換します。
flagsにはローマ字変換の方法を記述します。

kunrei訓令式ローマ字を使用する
dzu「づ」をdzuとする
oh長音をhで表現する
macron長音をマクロンで表現する
circumflex長音をサーカムフレックスで表現する
quotenの後に母音やヤ行が続くときにシングルクオートを使用する
hyphennの後に母音やヤ行が続くときにハイフンを使用する
Function: romaji->katakana string
Function: romaji->katakana string flags …

ローマ字をカタカナに変換します。
flagsにはローマ字変換の方法を記述します。

kunrei訓令式ローマ字を使用する

5.44.4 ラテン語ユーティリティ

Function: normalize-classic-latin modern

文字列modernを古典期のラテン語表現に変更します。W=VVとして扱います。

(normalize-classic-latin "Jupiter") -> "IVPITER"
(normalize-classic-latin "cujus") -> "CVIVS"
(normalize-classic-latin "wples") -> "VVLPES"
(normalize-classic-latin "降臨せよJupiter!") -> "降臨せよIVPITER!"
Function: latin-string=? string1 …

string1がラテン語の文字として等しいかを比較します。 

(latin-string=? "Jupiter" "Ivpiter") -> #t
(latin-string=? "cujus" "cvivs") -> #t
(latin-string=? "wlpes" "vulpes") -> #t
(latin-string=? "Jupiter" "IVPITER") -> #f
Function: latin-string-ci=? string1 …

string1が大文字小文字を区別せずラテン語の文字として等しいかを比較します。 

(latin-string-ci=? "Jupiter" "Ivpiter") -> #t
(latin-string-ci=? "cujus" "cvivs") -> #t
(latin-string=? "WLPES" "vulpes") -> #t
(latin-string-ci=? "Jupiter" "IVPITER") -> #t

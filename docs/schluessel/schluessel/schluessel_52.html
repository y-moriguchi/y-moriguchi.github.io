5.14 LDAPアクセス

LDAPサーバにアクセスする手続きです。

Function: bind-ldap-service hostname dc password

LDAPサーバと接続します。
hostnameはホスト名、dcはLDAP管理者のdc、passwordはパスワードです。
例を以下に示します。 

(define ldap-connection
  (bind-ldap-service
     "hostname"  ; ホスト名
     "cn=Manager,dc=example,dc=com"  ; dc
     "secret"))  ; パスワード
Function: select-ldap-service ldap-connection dn cond level

LDAPサーバから情報を取得します。
ldap-connectionはbind-ldap-serviceで生成したLDAP接続、 dnは検索対象のdnです。

condは以下の文法で表される式です。

cond-expression
: (= attr value) ; 等しい
| (~= attr value) ; ほとんど等しい
| (<= attr value) ; 小さい
| (>= attr value) ; 大きい
| (and cond-expression …)
| (or cond-expression …)
| (not cond-expression)

levelは以下のシンボルを指定します。

object対象オブジェクトまで
1level1レベルまで
subtreeサブツリーまで

例は以下の通りです。 

(select-ldap-service
  ldap-connection  ; バインドしたLDAP接続
  "ou=User,dc=example,dc=com"  ; 検索するdn
  '(or (= "cn" "User*")
       (= "sn" "aaa*"))  ; 検索条件
  'subtree)  ; object, 1level, subtreeが指定可能
Function: insert-ldap-service! ldap-connection dn value

LDAPサーバに情報を登録します。
ldap-connectionはbind-ldap-serviceで生成したLDAP接続、 dnは検索対象のdnです。

valueは以下の形式で表されるリストです。 (("attr" "value1" …)
("attr" "value2" …)
...)

例は以下の通りです。 

(insert-ldap-service!
  ldap-connection  ; バインドしたLDAP接続
  "cn=User0001,ou=User,dc=example,dc=com"
  '(("cn" "User0001")
    ("sn" "aaaaa")
    ("objectclass" "top" "person" "organizationalPerson")))
Function: update-ldap-service! ldap-connection dn value

LDAPサーバの情報を変更します。
ldap-connectionはbind-ldap-serviceで生成したLDAP接続、 dnは検索対象のdnです。

valueは以下の形式で表されるリストです。 ((replace "attr" "value1" …) ; 変更
(add "attr" "value2" …) ; 追加
(remove "attr") ; 削除
...)

例は以下の通りです。 

(update-ldap-service!
  ldap-connection  ; バインドしたLDAP接続
  "cn=User0001,ou=User,dc=example,dc=com"
  '((replace "sn" "bbbb")
    (add     "description" "A description")
    (remove "mail")))
Function: delete-ldap-service! ldap-connection dn

LDAPサーバの情報を削除します。
ldap-connectionはbind-ldap-serviceで生成したLDAP接続、 dnは検索対象のdnです。

例は以下の通りです。 

(delete-ldap-service!
  ldap-connection  ; バインドしたLDAP接続
  "cn=User0001,ou=User,dc=example,dc=com")

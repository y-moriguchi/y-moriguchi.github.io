5.3 イテレータ

リストのように扱えるオブジェクトです。

通常、リストを処理するときには以下のような記述をします。 

(define (foo lst)
  (cond ((null? lst) #f)
        ((pred? (car lst)) <処理>)
        (else (foo (cdr lst)))))

リストに限らず、このような処理ができるオブジェクトをまとめることにより、 統一的にそのオブジェクトをまとめることができます。

イテレータを使用した場合、上の例は 

(define (foo lst)
  (cond ((iterator-null? lst) #f)
        ((pred? (iterator-car lst)) <処理>)
        (else (foo (iterator-cdr lst)))))

のように記述できます。

リストはそのままイテレータとして扱えます。

"イテレート可能なオブジェクト"は手続き->iteratorにより イテレータを取り出すことが可能なオブジェクトです。

イテレート可能なオブジェクトにはリスト、文字列、ベクタがあります。

Function: iterator? obj

オブジェクトがイテレータのとき#tを得ます。

Function: iterable? obj

オブジェクトがイテレート可能なオブジェクトのとき#tを得ます。

Function: iterator-null? iterator

イテレータが終端にあるとき#tを得ます。

Function: iterator-car iterator

イテレータが指し示すオブジェクトを取得します。

Function: iterator-cdr iterator

イテレータを次に進めます。

Function: ->iterator iterable

イテレート可能なオブジェクトからイテレータを取得します。

Function: iterator->list iterator

イテレータをリストに変換します。

Function: compound-iterator iterator …

イテレータを連結します。
例えば、イテレータitr1、itr2、itr3の内容が{1, 2, 3、{4, 5、{6, 7であるとしたとき、 イテレータ(compound-iterator itr1 itr2 itr3)の内容は{1, 2, 3, 4, 5, 6, 7となります。

Function: filtered-iterator proc iterator

(proc (iterator-car iterator))が#tになった要素のみイテレートします。
例えば、イテレータitrの内容が{1, 2, 3, 4であるとき、 イテレータ(filtered-iterator even? itr)の内容は{2, 4となります。

Function: group-iterator iterator …

それぞれのイテレータの結果をリストに連結して取得します。
例えば、イテレータitr1、itr2、itr3の内容が{1, 2, 3、{4, 5、{6, 7であるとしたとき、 イテレータ(group-iterator itr1 itr2 itr3)の内容は{(1 4 6), (2 5 7)となります。

Function: list-group-iterator list …

それぞれのリストのcarの結果をリストに連結して取得します。
例えば、リストitr1、itr2、itr3の内容が(1 2)、(4 5)、(6 7)であるとしたとき、 イテレータ(group-iterator itr1 itr2 itr3)の内容は{(1 4 6), (2 5 7)となります。
リストの長さが異なる、正確なリストでない、またはリストが循環するとき、 該当したところが出現したときにエラーとなります。

Function: make-csv-iterator file [separator] [quote] [newline] [comment]

CSVを解析するイテレータを取得します。
separatorはデフォルトで’,’、
quoteはデフォルトで’"’、
newlineはデフォルトで’\n’、
commentはデフォルトで#fです。

Function: make-real-number-iterator start end step

startで始まり、step間隔でendまで増えて(減って)いくイテレータを取得します。

Function: make-string-tokenizer string charset

stringをcharsetの文字で区切り、その結果を取得するイテレータを取得します。
charsetには文字セットまたは文字列が設定可能です。

Function: make-tree-walker tree

treeの要素をトラバースするイテレータを取得します。
例えば、treeを((1 2) 3 ((4 5) 6))としたとき、 イテレータの内容は{1, 2, 3, 4, 5, 6となります。

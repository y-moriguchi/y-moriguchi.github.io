5.32.1 メッセージダイジェスト

Function: string->message-digest string algorithm

メッセージダイジェストオブジェクトを生成します。
algorithmによりダイジェストの生成アルゴリズム(md5、sha等)を指定します。

Function: string->hexified-message-digest string algorithm

メッセージダイジェストを16進化した文字列を生成します。

Function: file->message-digest string algorithm

ファイルからメッセージダイジェストオブジェクトを生成します。

Function: file->hexified-message-digest string algorithm

ファイルからメッセージダイジェストを16進化した文字列を生成します。

Function: hexift-message-digest digest

メッセージダイジェストを16進化します。

5.32.2 オブジェクトの署名

Function: generate-key-pair algorithm

指定したアルゴリズム(rsa等)により公開鍵・秘密鍵の対を生成します。
戻り値は公開鍵、秘密鍵の2値を返します。

Function: sign-file path private-key

指定したパスのファイルを署名し、署名を返します。

Function: file-signed? path public-key sign

指定されたファイルの署名を確認し、署名されていれば#fを返します。

Function: sign-file obj private-key

指定したオブジェクトを署名し、署名されたオブジェクトを返します。

Function: verify-signed-datum obj public-key

指定された署名されたオブジェクトを検証し、正しければ#tを得ます。

Function: unwrap-signed-datum obj

署名されたオブジェクトをアンラップします。

Function: hexify-signeture sign

署名を16進化します。

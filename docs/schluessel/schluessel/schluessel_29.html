4.21 入出力

Function: port? object

[R5RS] objectがポートのとき#tを返します。

Function: input-port? object

[R5RS] objectが入力ポートのとき#tを返します。

Function: output-port? object

[R5RS] objectが出力ポートのとき#tを返します。

Function: current-input-port

[R5RS] 現在の入力ポートを取得します。

Function: current-output-port

[R5RS] 現在の出力ポートを取得します。

Function: open-input-file filename

[R5RS] filenameで表されるファイルを入力用に開きます。
ファイルはテキストモードで開かれます。 バイナリファイルを使用したいときは R6RS仕様のライブラリを使用してください。

Function: open-output-file

[R5RS] filenameで表されるファイルを出力用に開きます。
ファイルはテキストモードで開かれます。 バイナリファイルを使用したいときは R6RS仕様のライブラリを使用してください。

Function: call-with-input-file filename proc

[R5RS] filenameで表されるファイルを入力用に開き、 それをprocの引数として呼び出します。
procは1引数の手続きである必要があります。

Function: call-with-output-file filename proc

[R5RS] filenameで表されるファイルを出力用に開き、 それをprocの引数として呼び出します。
procは1引数の手続きである必要があります。

Function: close-input-port port

[R5RS] 入力ポートをクローズします。

Function: close-output-port port

[R5RS] 出力ポートをクローズします。

Function: read port

[R5RS] ポートからS式1つを読み込みます。

Function: read-char port

[R5RS] ポートから文字を1つ読み込みます。

Function: peek-char port

[R5RS] ポートの先頭の文字を1つ読み込んで戻します。

Function: eof-object? object

[R5RS] objectがファイル終端(EOF)オブジェクトのとき#tを返します。

Function: char-ready? port

[R5RS] portからブロックせずに読み出せるとき#tを返します。

Function: write object [port]

[R5RS] 出力ポートにS式を出力します。 ポートが省略されたときは標準出力に出力します。
コンピュータよりの形式で出力します。

Function: display object [port]

[R5RS] 出力ポートにS式を出力します。 ポートが省略されたときは標準出力に出力します。
人間よりの形式で出力します。

Function: write-char char [port]

[R5RS] ポートに文字を出力します。 ポートが省略されたときは標準出力に出力します。

Function: newline [port]

[R5RS] ポートに改行を出力します。 ポートが省略されたときは標準出力に出力します。

Function: with-input-from-file filename thunk

[R5RS] filenameで表されるファイルを入力用に開き、 それを現在の入力ポートとしてthunkを呼び出します。
thunkは0引数の手続きである必要があります。

Function: with-output-to-file filename thunk

[R5RS] filenameで表されるファイルを出力用に開き、 それを現在の出力ポートとしてthunkを呼び出します。
thunkは0引数の手続きである必要があります。

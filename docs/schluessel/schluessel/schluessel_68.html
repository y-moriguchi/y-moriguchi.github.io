5.30 プロセス

プロセスを起動する手続きです。

Function: run-process (command …) env dir

プロセスを起動します。
commandは実行するコマンドをシンボルまたは文字列の形で与えます。
以下のシンボルは特殊な意味を持ちます。

> filenameファイルに標準出力をリダイレクトします
2> filenameファイルに標準エラー出力をリダイレクトします
>>ファイルに標準出力を追加でリダイレクトします
2>> filenameファイルに標準エラー出力を追加でリダイレクトします
2>&1 filename標準出力と標準エラー出力をマージします
cmd1 | cmd2cmd1の出力をcmd2の入力とします(パイプ)
Function: system cmd

環境で使用しているシェルによりコマンドを実行します。
UNIX系OSおよびMac OS Xではsh、bash、tcsh等が、 Windowsではcmd.exeが(標準設定では)起動されます。

4.8 シンボル

Function: symbol? object

[R5RS] objectがシンボルのときに#tを返します。

Function: symbol->string symbol

[R5RS] symbolを文字列に変換します。

Function: string->symbol string

[R5RS] stringをシンボルに変換します。
string->symbolで生成されたシンボルはシンボル名が大文字小文字を 区別しない設定になっていても大文字小文字を区別します。

Function: symbol=? bool …

[R6RS] 与えられたシンボルがすべて等しいとき#tを返します。

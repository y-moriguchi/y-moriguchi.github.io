5.21 数学(定数)

数学定数です。定数は超越数も含めて厳密な値を持ちます。 (近似値ではありません)

Variable: pi

円周率\piを厳密に表します。

Variable: pi/180

\pi/180を厳密に表します。

Variable: e

自然対数の底eを厳密に表します。

Variable: golden-number

黄金比を厳密に表します。

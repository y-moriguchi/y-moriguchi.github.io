3.1.1 数の階層

数は部分集合による階層を持ちます。各階層に対応する述語があります。 

数(number?)
八元数(octonion?)
四元数(quaternion?)
複素数(complex?)
実数(real?)
有理数(rational?)
整数(integer?)

例えば、72は整数、有理数、実数、複素数、四元数、八元数、そして数です。 以下の例はすべて#tとなります。 

(number? 72)
(octonion? 72)
(quaternion? 72)
(complex? 72)
(real? 72)
(rational? 72)
(integer? 72)

3.1.2 正確な数と不正確な数

数には正確な数と不正確な数があります。

正確な数は、数学的に厳密に表される数をさします。 Schluesselで表される正確な数の例として、整数(例: 72)、分数(72/91)、 厳密な数学定数、厳密な数よりなる複素数・四元数・八元数があります。

不正確な数は、浮動小数点などで精度が制限される数を指します。 不正確な数の例として、浮動小数点数(例: 72.0, 72.9)、 不正確な数よりなる複素数・四元数・八元数があります。

正確な数の演算は可能な限り正確な数になるように計算します。
例: 

(+ 72 9) -> 81
(+ 72.0 9) -> 81.0
(sqrt 961) -> 31
(sqrt 961.0) -> 31.0
(sqrt 765) -> 27.65863337187866

3.1.3 無限大と非数

Schluesselにおいて不正確な実数の無限大は+inf.0, -inf.0で表されます。 また、不正確な複素数の(リーマン球における)無限遠点は+inf.0@(任意の数)で 表されます。

複素数において、無限遠点と思われる表記には+inf.0+inf.0iのような 直交座標において要素が無限遠になったものと +inf.0@nがありますが、それらは等価ではありません。

非数はnan.0で表されます。無限と非数の演算は次のようになります。 

(+ nan.0 1) -> nan.0
(+ nan.0 +inf.0) -> nan.0
(+ inf.0 +inf.0) -> inf.0
(- inf.0 +inf.0) -> nan.0
(/ 1.0 0.0) -> +inf.0
(/ 1 0) -> エラー

3.1.4 -0.0の区別

Schluesselでは-0.0を区別します。 

(/ 1.0 +0.0) -> +inf.0
(/ 1.0 -0.0) -> -inf.0

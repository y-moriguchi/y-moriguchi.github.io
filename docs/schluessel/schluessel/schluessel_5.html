2.3 システムプロパティ

Schluesselの起動前に以下のシステムプロパティを指定することにより動作を変更することができます。オプションによる指定はシステムプロパティによる指定に優先します。

System Property: schluessel.locale=locale

メッセージのロケールを指定します。

System Property: schluessel.r5rs=boolean

trueのときR5RSのみを使用します。

System Property: schluessel.import=package,package...

インポートするパッケージを指定します

System Property: schluessel.foldcase=boolean

trueのとき大文字・小文字を区別します

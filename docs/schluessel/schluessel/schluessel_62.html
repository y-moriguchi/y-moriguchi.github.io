5.24 1変数の多項式

1変数の多項式を計算します。

Function: polynomial1 n …

戻り値: 1変数多項式
1変数の多項式を生成します。

(polynomial1 1 2 3 4)

は多項式x^3+2x^2+3x+4を表します。

Function: polynomial1? obj

オブジェクトが1変数の多項式ならば#tを返します。

Function: differenciate1 poly

戻り値: 1変数多項式
多項式を微分します。

Function: integrate1 poly begin end

戻り値: 数値
多項式の区間[begin end]での定積分を求めます。

Function: substitute1 poly x

戻り値: 数値
多項式にxを代入した値を求めます。

Function: + n …

戻り値: 1変数多項式
[R5RS+] 多項式の和を求めます。この手続きは多項式に拡張されています。

Function: - n …

戻り値: 1変数多項式
[R5RS+] 多項式の差を求めます。この手続きは多項式に拡張されています。

Function: * n …

戻り値: 1変数多項式
[R5RS+] 多項式の積を求めます。この手続きは多項式に拡張されています。

Function: quotient n m

戻り値: 1変数多項式
[R5RS+] 多項式の商を求めます。この手続きは多項式に拡張されています。

Function: remainder n m

戻り値: 数値
[R5RS+] 多項式の剰余を求めます。この手続きは多項式に拡張されています。

Function: degree-polynomial1 poly

戻り値: 整数
多項式の次数を求めます。0の次数は-1です。

Function: integrate-polynomial1 poly C

戻り値: 1変数多項式
多項式を不定積分します。積分定数はCで与えます。

Function: polynomial1-ref poly n

戻り値: 数値
多項式のn次の項の数値を返します。 次数が多項式の次数を超えたときには0を返します。

Function: solve-polynomial1 polynomial1

方程式c_nx^n+c_{n-1}x^{n-1}+\dots+c_0=0を解き、 その全ての実数解をリストの形で返します。
値は不正確な数で得られます。

Function: solve-polynomial1-by-bairstow polynomial1

方程式c_nx^n+c_{n-1}x^{n-1}+\dots+c_0=0をBairstow法で解き、 その全ての複素数解をリストの形で返します。
値は不正確な数で得られます。
反復法なので値が収束しないこともあります。

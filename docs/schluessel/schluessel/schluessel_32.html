4.24 R6RSハッシュテーブル

Function: make-hashtable hash-function equiv
Function: make-hashtable hash-function equiv k

[R6RS]ハッシュ関数手続きhash-functionと等値手続きequivをもつハッシュテーブルを構築します。kでハッシュテーブルの初期容量を指定します。

Function: make-eq-hashtable
Function: make-eq-hashtable k
Function: make-eqv-hashtable
Function: make-eqv-hashtable k

[R6RS]それぞれeq?,eqv?で比較されるハッシュテーブルを作成します。kでハッシュテーブルの初期容量を指定します。

Function: hashtable? obj

[R6RS]objがハッシュテーブルであれば#tを返します。

Function: hashtable-size hashtable

[R6RS]hashtableのサイズを返します。

Function: hashtable-ref hashtable key
Function: hashtable-ref hashtable key default

[R6RS+]hashtableのkeyに対応する値を返します。 keyに対応する値がないときはdefaultを得ます。 defaultが存在しないときは未定義値を返します。

Function: hashtable-set! hashtable key obj

[R6RS]hashtableのkeyに対応する値としてobjを登録します。

Function: hashtable-delete! hashtable key

[R6RS]hashtableのkeyに対応する値を削除します。

Function: hashtable-contains? hashtable key

[R6RS]hashtableのkeyに対応する値があれば#tを返します。

Function: hashtable-update! hashtable key proc default

[R6RS]hashtableのkeyに対応する値があればその値に、なければdefaultに対して手続きprocを適用し、その戻り値をkeyに対応する値として変更します。
この手続きは以下のコードと等価です。 

(hashtable-set!
  hashtable
  key
  (proc (hashtable-ref hashtable key default)))
Function: hashtable-copy hashtable
Function: hashtable-copy hashtable mutable

[R6RS]hashtableのコピーを返します。 mutableがあって引数が#tのときは戻り値のハッシュテーブルは変更可能です。それ以外のときは変更不可能です。

Function: hashtable-clear! hashtable
Function: hashtable-clear! hashtable k

[R6RS]hashtableをクリアします。 kが与えられたときは容量がkに再設定されます。

Function: hashtable-keys hashtable

[R6RS]hashtableのキーをすべてベクタにして返します。

Function: hashtable-entries hashtable

[R6RS]hashtableのキーとそれに対応する値をすべてベクタにしたものの2値を返します。

Function: hashtable-equivalence-function hashtable

[R6RS]hashtableのキーの等値比較に使用されている手続きを返す。

Function: hashtable-hash-function hashtable

[R6RS]hashtableのキーのハッシュ値生成に使用されている手続きを返す。

Function: hashtable-mutable? hashtable

[R6RS]hashtableが変更可能なときに#tを返す。

Function: equal-hash obj

[R6RS]objに対して、その構造と内容に基づいたハッシュ値を返す。

Function: string-hash string

[R6RS]文字列stringに対して、その内容に基づいたハッシュ値を返す。

Function: string-ci-hash string

[R6RS]文字列stringに対して、その内容に基づいたハッシュ値を返す。 大文字と小文字の区別をしない。

Function: symbol-hash symbol

[R6RS]シンボルsymbolに対して、その内容に基づいたハッシュ値を返す。

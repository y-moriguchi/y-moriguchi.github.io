5.26 線形代数ライブラリ

行列を操作する手続きです。

Function: list->matrix ((x …) …)

戻り値: 行列
リストから行列を生成します。
次の手続き呼び出しは 

(list->matrix '((1 2 3) (4 5 6)))

行列 を返します。

Function: matrix? obj

戻り値: objが行列のとき#t
オブジェクトが行列型であるかを判定します。

Function: list->number-vector (x …)

戻り値: 数値ベクトル
リストから数値ベクトルを生成します。
次の手続き呼び出しは 

(list->number-vector '(1 2 3))

ベクトル(1 \ 2 \ 3)を返します。

Function: number-vector x …

戻り値: 数値ベクトル
引数から数値ベクトルを生成します。
次の手続き呼び出しは 

(number-vector 1 2 3)

ベクトル(1 \ 2 \ 3)を返します。

Function: make-matrix rowsize colsize [fill]

戻り値: 行列
一様な値を持つ行列を生成します。fillのデフォルト値は0とします。
次の手続き呼び出しは 

(make-matrix 3 4 1)

行列 を返します。

Function: make-number-vector size [fill]

戻り値: 数値ベクトル
一様な値を持つ数値ベクトルを生成します。
次の手続き呼び出しは 

(make-number-vector 3 1)

ベクトル(1 \ 1 \ 1)を返します。

Function: matrix-determinant matrix

戻り値: 数値
行列の行列式を求めます。

Function: matrix= matrix1 matrix2

戻り値: 行列が等しいとき#t
行列が等しいかを調べます。数の比較には=を使用します。

Function: matrix+ matrix …

戻り値: 行列
行列の和を求めます。

Function: matrix- matrix …

戻り値: 行列
行列の差を求めます。

Function: matrix* matrix …

戻り値: 行列
行列の積を求めます。

Function: matrix-invert matrix …

戻り値: 行列
逆行列を求めます。

Function: lu-decompose matrix

戻り値: 置換行列、L行列、U行列の3値
行列をLU分解し、P(LU)の形にします。

Function: solve-linear-equation matrix vector

戻り値: 連立1次方程式の解
方程式A\vec{x}=\vec{b}を解きます。

Function: matrix-row-size matrix
Function: matrix-column-size matrix

戻り値: 整数
行列matrixの行(列)のサイズを返します。

Function: matrix-row-vector matrix n
Function: matrix-column-vector matrix n

戻り値: ベクトル
行列matrixのn番目の行(列)ベクトルを返します。

Function: matrix-transpose matrix

戻り値: 転置行列A^t
行列matrixの転置行列を返します。

2.1 Schluesselの起動

SchluesselはJARファイルで提供されています。

Schluesselはcom.sunなどのパッケージを一切使用していないのでJava SE 6がサポートされているJava VMの実装なら動作すると思います。

Schluesselを起動するには、コマンドライン(Windowsならばコマンドプロンプト、UNIX系OSならば端末)から 

$ java -jar schluesse.jar [オプション] [-|file]

とします。

すべての引数が省略されたときはREPLが起動します。

ファイル名に"-"を指定すると、標準入力からスクリプトを読み込みそのまま終了します。

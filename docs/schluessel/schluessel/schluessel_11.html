4.3 引用

Special Form: quote object

[R5RS] objectそのものを返します。

Special Form: quasiquote object

[R5RS] objectを準クオートします。 マクロの定義やSwingのレイアウト定義などに有用です。

Special Form: unquote object
Special Form: unquote-splicing object

[R5RS] objectをアンクオートします。 quasiquoteの内部のみで有効です。

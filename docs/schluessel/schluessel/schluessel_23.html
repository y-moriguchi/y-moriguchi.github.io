4.15 文字列

Function: string? object

[R5RS] objectが文字列のときに#tを返します。

Function: string<? string …
Function: string<=? string …
Function: string>? string …
Function: string>=? string …
Function: string=? string …

[R5RS] それぞれ左から右へ文字列がより小さい、以下である、より大きい、 以上であり、等しいときに#tを返します。 文字列は辞書順で比較されます。

Function: string-ci<? string …
Function: string-ci<=? string …
Function: string-ci>? string …
Function: string-ci>=? string …
Function: string-ci=? string …

[R5RS] それぞれ左から右へ文字列がより小さい、以下である、より大きい、 以上であり、等しいときに#tを返します。 文字列は辞書順で比較されます。
これらのメソッドは大文字小文字を区別しません。

Function: make-string k [char]

[R5RS] k文字からなる文字列を作成します。 charが与えられればそれで内容を満たします。 charが与えられなければ空白文字で満たされます。

Function: string-length string

[R5RS] stringの長さを取得します。

Function: string-ref string k

[R5RS] stringのk文字目の文字を取得します。

Function: string-copy string [start end]

[R5RS] stringをコピーします。 start(その場所を含む)、end(その場所を含まない)で 範囲を指定することもできます。

Function: string-append string …

[R5RS] 文字列を連結します。

Function: substring string start end

[R5RS] start(その場所を含む)からend(その場所を含まない)までの 部分文字列を返します。

Function: string->list string

[R5RS] 文字列から文字のリストに変換します。

Function: list->string list-of-chars

[R5RS] 文字のリストから文字列に変換します。

Function: string-set! string k char

[R5RS] stringのk番目の文字をcharにセットします。
現在の仕様では文字列の内容を書き換えるたびに 内部で新しい文字列インスタンスを生成しています。 そのため、make-stringなどで文字列領域を割り当て、 string-set!で更新するアルゴリズムは効率が悪くなります。
(ただし、将来のバージョンでは変更される可能性があります。)

Function: string-fill! string char [start end]

[R5RS] stringの要素をすべてcharにします。 start(その場所を含む)、end(その場所を含まない)で 範囲を指定することもできます。

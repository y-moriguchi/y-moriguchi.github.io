4.9 リスト

Function: cons obj1 obj2

[R5RS] コンス対(obj1 . obj2)を生成します。

Function: car pair

[R5RS] pairのcar値を取り出します。

Function: cdr pair

[R5RS] pairのcdr値を取り出します。

Function: caar pair
Function: cadr pair
Function: cdar pair
Function: cddr pair
Function: caaar pair
Function: caadr pair
Function: cadar pair
Function: caddr pair
Function: cdaar pair
Function: cdadr pair
Function: cddar pair
Function: cdddr pair
Function: caaaar pair
Function: caaadr pair
Function: caadar pair
Function: caaddr pair
Function: cadaar pair
Function: cadadr pair
Function: caddar pair
Function: cadddr pair
Function: cdaaar pair
Function: cdaadr pair
Function: cdadar pair
Function: cdaddr pair
Function: cddaar pair
Function: cddadr pair
Function: cdddar pair
Function: cddddr pair

[R5RS] 複合したcar、cdrの短縮形です。 例えば、cddadrは以下の手続きと同じになります。 

(cdr (cdr (car (cdr x))))
Function: null? object

[R5RS] objectが空リストのときに#tを返します。

Function: pair? object

[R5RS] objectがコンス対のときに#tを返します。

Function: list? object

[R5RS] objectが正しいリストのときに#tを返します。 コンス対であっても、

  • リスト終端のcdrが空リストでない
  • リストが循環している

のときには#fを返します。

Function: list object …

[R5RS] リスト(object …)を取得します。

Function: length list

[R5RS] リストの長さを取得します。

Function: list-ref list k

[R5RS] listからk番目の要素を取得します。

Function: list-tail list k

[R5RS] listからk番目のcdrを返します。

Function: append list …

[R5RS] list…をつなげたリストを返します。

Function: reverse list

[R5RS] listを逆順にしたリストを返します。

Function: memq object list
Function: memv object list
Function: member object list

[R5RS] listからobjectを探し、 一致したところとそれ以降のリストを返します。 一致しないときには#fを返します。

memq、memv、memberはオブジェクトの比較に それぞれeq?、eqv?、equal?を使用します。

Function: assq object alist
Function: assv object alist
Function: assoc object alist

[R5RS] alistは連想リスト (コンス対でkey-valueペアを格納しているリスト)である必要があります。

alistのcar部とobjectが一致するコンス対を探し、 一致したところのコンス対を返します。 一致しないときには#fを返します。

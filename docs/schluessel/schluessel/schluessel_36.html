4.28 R5RSその他

Function: transcript-on filename
Function: transcript-off

[R5RS] R5RSのドキュメントを参照願います。

Function: load filename

[R5RS+] 拡張機能リファレンスを参照願います。

Function: exit error-code

[R6RS] error-codeでSchluesselを終了します。 error-codeが#fのときは0以外のコードで終了します。

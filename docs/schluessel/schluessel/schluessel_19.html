4.11 算術

Function: number? object

[R5RS] objectが数のとき#tを返します。

Function: complex? object

[R5RS] objectが複素数のとき#tを返します。

Function: real? object

[R5RS] objectが実数のとき#tを返します。 虚数部分が不正確な0のときは#fを返します。

Function: rational? object

[R5RS] objectが有理数のとき#tを返します。 虚数部分が不正確な0のときは#fを返します。

Function: integer? object

[R5RS] objectが整数のとき#tを返します。 虚数部分が不正確な0のときは#fを返します。

Function: real-valued? object

[R6RS] objectが実数のとき#tを返します。 虚数部分が不正確な0のときでも#tを返します。

Function: rational-valued? object

[R6RS] objectが有理数のとき#tを返します。 虚数部分が不正確な0のときでも#tを返します。

Function: integer-valued? object

[R6RS] objectが整数のとき#tを返します。 虚数部分が不正確な0のときでも#tを返します。

Function: + number …

[R5RS+] 加算を実行します。 この手続きは体や環などの代数的な対象に拡張されています。

Function: - number …

[R5RS] 減算を実行します。 この手続きは体や環などの代数的な対象に拡張されています。

Function: * number …

[R5RS] 乗算を実行します。 この手続きは体や環などの代数的な対象に拡張されています。

Function: / number …

[R5RS] 除算を実行します。 この手続きは体などの代数的な対象に拡張されています。

Function: < number …
Function: <= number …
Function: > number …
Function: >= number …
Function: = number …

[R5RS+] それぞれ左から右へ数値がより小さい、以下である、より大きい、 以上であり、等しいときに#tを返します。 この手続きは数と見なせる対象に拡張されています。

Function: exact? number

[R5RS] numberが正確な数のとき#tを返します。

Function: inexact? number

[R5RS] numberが不正確な数のとき#tを返します。

Function: zero? number

[R5RS+] numberが0のとき#tを返します。 この手続きは代数的に零元と見なせるものについて #tを返すよう拡張されています。

Function: positive? number

[R5RS] numberが正数のとき#tを返します。

Function: negative? number

[R5RS] numberが負数のとき#tを返します。

Function: even? number

[R5RS+] numberが偶数のとき#tを返します。 この手続きはパリティが偶と見なせるものについて #tを返すよう拡張されています。

Function: odd? number

[R5RS+] numberが奇数のとき#tを返します。 この手続きはパリティが奇と見なせるものについて #tを返すよう拡張されています。

Function: finite? number
Function: infinite? number

[R6RS] numberが有限(無限)のとき#tを返します。

Function: nan? number

[R6RS] numberがNaNのとき#tを返します。

Function: quotient number div

[R5RS+] numberを整除した結果を返します。 この手続きは整除できる代数的対象について拡張されています。

Function: remainder number div

[R5RS+] numberを整除した余りを返します。 この手続きは整除できる代数的対象について拡張されています。

Function: modulo number div

[R5RS] numberを整除したモジュロを返します。 quotientとの違いについてはR5RSのドキュメントを参照願います。

Function: div number div

[R6RS] numberを整除した結果を返します。 余りは必ず0以上になります。

Function: mod number div

[R6RS] numberを整除した余りを返します。 余りは必ず0以上になります。

Function: div-and-mod number div

[R6RS] numberを整除した結果と余りを返します。 余りは必ず0以上になります。

Function: div0 number div

[R6RS] numberを整除した結果を返します。 整数についてquotientと同じ結果を返しますが、 整除できる代数的対象について拡張されていません。

Function: mod0 number div

[R6RS] numberを整除した余りを返します。 整数についてremainderと同じ結果を返しますが、 整除できる代数的対象について拡張されていません。

Function: div0-and-mod0 number div

[R6RS] numberを整除した結果と余りを返します。 整数についてquotientとremainderと同じ結果を返します。

Function: truncate number

[R5RS] numberの小数点を切り捨てた結果を返します。

Function: floor number

[R5RS] numberを超えない最大の整数を返します。

Function: ceiling number

[R5RS] numberを超える最小の整数を返します。

Function: round number

[R5RS] numberを四捨五入した整数を返します。

Function: abs number

[R5RS] numberの絶対値を返します。

Function: exact->inexact number
Function: inexact->exact number

[R5RS] 正確な数から不正確な数、およびその逆に変換します。

Function: inexact number
Function: exact number

[R6RS] 正確な数から不正確な数、およびその逆に変換します。

Function: max number …
Function: min number …

[R5RS] number…のうちで最大(最小)の値を返します。

Function: gcd number …

[R5RS] number…の最大公約数を返します。

Function: lcm number …

[R5RS] number…の最小公倍数を返します。

Function: rationalize number number2

[R5RS] 詳細はR5RSのドキュメントを参照願います。

Function: numerator rational
Function: denominator rational

[R5RS] rationalの分子(分母)を取得します。

Function: real-part number
Function: imag-part number

[R5RS] numberの実部(虚部)を取得します。

Function: make-rectangular real-part imag-part

[R5RS] 直交座標による複素数を生成します。

Function: make-polar radius angle

[R5RS] 極座標による複素数を生成します。

Function: magnitude complex
Function: angle complex

[R5RS] 複素数の半径(偏角)を取得します。

Function: string->number string [radix]

[R5RS] 文字列を数値にパースします。 パースできない文字のときは#fを返します。
不正確な数の場合、radixは10のみ有効です。

Function: number->string number [radix [precision]]

[R5RS, R6RS] 数値を文字列に変換します。
不正確な数の場合、radixは10のみ有効です。

5.38 AWT関連

Java AWTに関連した手続きです。

現在、GUIコンポーネントの機能はSwingに置き換えられていますが、低レベルなGUIの操作や、トレイアイコンなど重量コンポーネントが必要なところでは使用されています。

5.38.1 デスクトップ操作

デスクトップに関連づけられているアプリケーションを起動する手続きです。

Function: desktop-supported? [operation]

実行している環境でデスクトップがサポートされていれば#tを返します。
operationは以下のシンボルで表される操作がサポートされているときに#tを返します。

browseWebブラウザの起動
edit編集
mailメーラの起動
open開く
print印刷
Function: desktop-browse uri
Function: desktop-edit file
Function: desktop-mail [uri]
Function: desktop-open file
Function: desktop-print file

それぞれWebブラウザの起動、編集、メーラの起動、開く、印刷する動作をします。 動作が実行されたときは#tを、動作がサポートされないときは#fが返されます。

5.38.2 グラフィックス環境

グラフィックス環境を操作する手続きです。フレームをフルスクリーンで実行することもできます。

Function: graphics-all-fonts

実行している環境でサポートされている全フォントのオブジェクトをリストの形で返します。

Function: graphics-default-screen-device

実行している環境でのデフォルトのスクリーンデバイスオブジェクトを返します。

Function: graphics-device-full-screen-set! [screen-device] frame

指定したスクリーンデバイスにフレームをフルスクリーン表示します。 戻り値は未定義です。

Function: graphics-screen-devices

実行している環境でサポートされている全スクリーンデバイスのオブジェクトを リストの形で返します。

Function: graphics-device-full-screen? [screen-device]

指定したスクリーンデバイスがフルスクリーン表示されていれば#tを返します。 そうでなければ#fを返します。

Function: graphics-device-full-screen-supported? [screen-device]

指定したスクリーンデバイスがフルスクリーン表示をサポートしていれば#tを返します。 そうでなければ#fを返します。

Function: graphics-headless?

実行している環境がグラフィックス環境を持たないとき#tを返します。

5.38.4 トレイアイコン

トレイアイコンに格納するアプリケーションを記述する手続きです。

Function: tray-supported?

トレイアイコンがサポートされていれば#tを返します。

Function: make-tray-icon image-file-path/image-object tooptip/#f menu/#f [action] [mouse-action] [mouse-motion-action]

トレイアイコンを作成します。
第1の引数はアイコンのイメージをファイルまたは画像オブジェクトで指定します。
第2の引数はツールチップを指定します。#fのときは表示されません。
第3の引数はメニューを指定します。#fのときはメニューは使用しません。 メニューは以下のように指定します。 

`(("menu1"  . ,(lambda (e) action))  ; 単純メニュー
  ("menusA"  ; ネストされたメニュー
    ("sub1" . ,(lambda (e) action))
    ("sub2" . ,(lambda (e) action))
    ...)
  ...)

第4〜第6の引数はアイコンがダブルクリックされた、マウス、マウスの移動に伴う アクションを記述します。
例を以下に示します。 

; トレイアイコンの定義
(define icon
  (make-tray-icon
    "example.png"
    "test"
    `(("sub"  ;メニュー階層
        (("A" ,(lambda (e) (display "a")))
         ("b" ,(lambda (e) (display "b")))))
      ("Exit" ,(lambda (e) (exit))))
    (lambda (e) (display "icon")) ;アクション
    `(;マウスアクション
      :entered ,(lambda (e) (display "entered"))
      :exited  ,(lambda (e) (display "exited")))))
Function: add-tray-icon! tray-icon

トレイアイコンをシステムトレイに登録します。

Function: display-tray-icon-message title message [type]

トレイアイコンにメッセージを送信します。
メッセージタイプは以下の通りです。デフォルトではinfoです。

errorエラー
warning警告
info情報
noneタイプなし

5.38.5 カーソルの形状変更

Function: cursor-set! componet [shape]

カーソルの形状を変更します。shapeは以下のようなシンボルまたは文字列を指定します。文字列を指定したときはネイティブのカーソル形状を使用することができます。省略されたときはデフォルトの形状が使用されます。

default-cursorデフォルトの形状
hand-cursor手の形状
crosshair-cursor十字の形状
move-cursor移動カーソル
n-resize-cursorサイズ変更(上)
s-resize-cursorサイズ変更(下)
e-resize-cursorサイズ変更(右)
w-resize-cursorサイズ変更(左)
ne-resize-cursorサイズ変更(右上)
nw-resize-cursorサイズ変更(左上)
se-resize-cursorサイズ変更(右下)
sw-resize-cursorサイズ変更(左下)
text-cursorテキストの形状
wait-cursor処理中の形状

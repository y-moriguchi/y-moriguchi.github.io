4.23 R6RSバイトベクタ

Special Form: endianness endianness-symbol

[R6RS]endianness-symbolが正しいかを判別し、正しいときにはそのシンボルを返します。 endianness-symbolは現在ではbigとlittleのみサポートしています。

Function: native-endianness

[R6RS]ネイティブのエンディアンに対応するシンボルを得ます。

Function: bytevector? obj

[R6RS]objがバイトベクタのとき#tを返します。

Function: make-bytevector k
Function: make-bytevector k fill

[R6RS]kバイトのバイトベクタを新たに割り当てます。 fill引数が指定されているときにはその値で初期化されます。 fillは-128から255の正確な整数である必要があります。

Function: bytevector-length bytevector

[R6RS]bytevectorの長さを返します。

Function: bytevector=? bytevector1 bytevector2

[R6RS]bytevector1とbytevector2の長さとバイト列が等しいときに#tを返します。

Function: bytevector-fill! bytevector fill
Function: bytevector-fill! bytevector fill start
Function: bytevector-fill! bytevector fill start end

[R6RS+]bytevectorのstartバイトからendバイトまでをfillの値で埋めます。 fillは-128から255の正確な整数である必要があります。 start,endは省略されたときそれぞれ0,bytevectorのサイズとなります。

Function: bytevector-copy! source source-start target target-start k

[R6RS]バイトベクタsourceのsource-startの位置からからバイトベクタtargetのtarget-startの位置までkバイト分コピーします。
source-start,target-start,kは非負の正確な整数で以下の条件を満たします。 

0 ≤ source-start ≤ source-start + k ≤ sourceの長さ
0 ≤ target-start ≤ target-target + k ≤ sourceの長さ
Function: bytevector-copy source
Function: bytevector-copy source start
Function: bytevector-copy source start end

[R6RS+]バイトベクタsourceのstartバイトからendバイトまでをコピーします。 start,endは省略されたときそれぞれ0,bytevectorのサイズとなります。

Function: bytevector-u8-ref bytevector k
Function: bytevector-s8-ref bytevector k

[R6RS]バイトベクタbytevectorからkバイト目の符号なし(bytevector-u8-ref)、符号付き(bytevector-s8-ref)の値それぞれを返します。

Function: bytevector-u8-set! bytevector k octet
Function: bytevector-s8-set! bytevector k byte

[R6RS]バイトベクタbytevectorからkバイト目に符号なし(bytevector-u8-set!)、符号付き(bytevector-s8-set!)の値をそれぞれをセットします。

Function: bytevector->u8-list bytevector
Function: u8-list->bytevector list

[R6RS]バイトベクタとオクテット(0から255)のリストを互いに変換します。

Function: bytevector-uint-ref bytevector k endianness size
Function: bytevector-sint-ref bytevector k endianness size

[R6RS]バイトベクタbytevectorからkバイトからバイト長sizeの符号なし(bytevector-uint-ref)、符号付き(bytevector-sint-ref)の値それぞれを返します。エンディアンはendiannessで指定します。

Function: bytevector-uint-set! bytevector k octet
Function: bytevector-sint-set! bytevector k byte

[R6RS]バイトベクタbytevectorからkバイトからバイト長sizeの符号なし(bytevector-uint-set!)、符号付き(bytevector-sint-set!)の値をそれぞれをセットします。エンディアンはendiannessで指定します。

Function: bytevector->uint-list bytevector endianness size
Function: uint-list->bytevector list endianness size
Function: bytevector->sint-list bytevector endianness size
Function: sint-list->bytevector list endianness size

[R6RS]整数のサイズがsizeでendiannessのエンディアンのバイトベクタとリストを互いに変換します。
bytevectorのサイズはsizeで割り切れる必要があります。

Function: bytevector-u16-ref bytevector k endianness
Function: bytevector-s16-ref bytevector k endianness
Function: bytevector-u32-ref bytevector k endianness
Function: bytevector-s32-ref bytevector k endianness
Function: bytevector-u64-ref bytevector k endianness
Function: bytevector-s64-ref bytevector k endianness

[R6RS]バイトベクタbytevectorからkバイトからの16ビット符号なし(bytevector-u16-ref)、16ビット符号付き(bytevector-s16-ref)、32ビット符号なし(bytevector-u32-ref)、32ビット符号付き(bytevector-s32-ref)、64ビット符号なし(bytevector-u64-ref)、64ビット符号付き(bytevector-s64-ref)の値それぞれを返します。エンディアンはendiannessで指定します。

Function: bytevector-u16-native-ref bytevector k
Function: bytevector-s16-native-ref bytevector k
Function: bytevector-u32-native-ref bytevector k
Function: bytevector-s32-native-ref bytevector k
Function: bytevector-u64-native-ref bytevector k
Function: bytevector-s64-native-ref bytevector k

[R6RS]bytevector-uxx-refと同じですが、ネイティブのエンディアンが使用されます。

Function: bytevector-u16-set! bytevector k n endianness
Function: bytevector-s16-set! bytevector k n endianness
Function: bytevector-u32-set! bytevector k n endianness
Function: bytevector-s32-set! bytevector k n endianness
Function: bytevector-u64-set! bytevector k n endianness
Function: bytevector-s64-set! bytevector k n endianness

[R6RS]バイトベクタbytevectorからkバイトに16ビット符号なし(bytevector-u16-ref)、16ビット符号付き(bytevector-s16-ref)、32ビット符号なし(bytevector-u32-ref)、32ビット符号付き(bytevector-s32-ref)、64ビット符号なし(bytevector-u64-ref)、64ビット符号付き(bytevector-s64-ref)の値それぞれをセットします。エンディアンはendiannessで指定します。

Function: bytevector-u16-native-set! bytevector k n
Function: bytevector-s16-native-set! bytevector k n
Function: bytevector-u32-native-set! bytevector k n
Function: bytevector-s32-native-set! bytevector k n
Function: bytevector-u64-native-set! bytevector k n
Function: bytevector-s64-native-set! bytevector k n

[R6RS]bytevector-uxx-set!と同じですが、ネイティブのエンディアンが使用されます。

Function: bytevector-ieee-half-ref bytevector k endianness
Function: bytevector-ieee-single-ref bytevector k endianness
Function: bytevector-ieee-double-ref bytevector k endianness
Function: bytevector-ieee-decimal32-ref bytevector k endianness
Function: bytevector-ieee-decimal64-ref bytevector k endianness

[R6RS+]バイトベクタbytevectorからkバイトからのIEEE形式の2進半精度(half)、2進単精度(single)、2進倍精度(double)、10進32ビット精度(decimal32)、10進64ビット精度(decimal64)、の値それぞれを返します。エンディアンはendiannessで指定します。

Function: bytevector-ieee-half-native-ref bytevector k
Function: bytevector-ieee-single-native-ref bytevector k
Function: bytevector-ieee-double-native-ref bytevector k
Function: bytevector-ieee-decimal32-native-ref bytevector k
Function: bytevector-ieee-decimal64-native-ref bytevector k

[R6RS+]bytevector-ieee-*-native-refと同じですが、ネイティブのエンディアンが使用されます。

Function: bytevector-ieee-half-set! bytevector k n endianness
Function: bytevector-ieee-single-set! bytevector k n endianness
Function: bytevector-ieee-double-set! bytevector k n endianness
Function: bytevector-ieee-decimal32-set! bytevector k n endianness
Function: bytevector-ieee-decimal64-set! bytevector k n endianness

[R6RS+]バイトベクタbytevectorのkバイトにIEEE形式の2進半精度(half)、2進単精度(single)、2進倍精度(double)、10進32ビット精度(decimal32)、10進64ビット精度(decimal64)、の値それぞれをセットします。エンディアンはendiannessで指定します。

Function: bytevector-ieee-half-native-set! bytevector k n
Function: bytevector-ieee-single-native-set! bytevector k n
Function: bytevector-ieee-double-native-set! bytevector k n
Function: bytevector-ieee-decimal32-native-set! bytevector k n
Function: bytevector-ieee-decimal64-native-set! bytevector k n

[R6RS+]bytevector-ieee-*-native-set!と同じですが、ネイティブのエンディアンが使用されます。

Function: string->utf8 string
Function: string->utf8 string start
Function: string->utf8 string start end

[R6RS+]文字列stringのstartからendまでのUTF-8表現のバイトベクタを得ます。
start,endは省略されたときそれぞれ0,stringの長さとなります。

Function: string->utf16 string
Function: string->utf16 string start
Function: string->utf16 string start end
Function: string->utf16 string endianness
Function: string->utf16 string endianness start
Function: string->utf16 string endianness start end

[R6RS+]文字列stringのstartからendまでのUTF-16表現のバイトベクタを得ます。
endiannessが指定されていないまたはbigのときはUTF-16BE、それ以外のときはUTF-16LEが使用されます。 start,endは省略されたときそれぞれ0,stringの長さとなります。

Function: string->utf32 string
Function: string->utf32 string start
Function: string->utf32 string start end
Function: string->utf32 string endianness
Function: string->utf32 string endianness start
Function: string->utf32 string endianness start end

[R6RS+]文字列stringのstartからendまでのUTF-32表現のバイトベクタを得ます。
endiannessが指定されていないまたはbigのときはUTF-32BE、それ以外のときはUTF-32LEが使用されます。 start,endは省略されたときそれぞれ0,stringの長さとなります。

Function: utf8->string bytevector
Function: utf8->string bytevector start
Function: utf8->string bytevector start end

[R6RS+]バイトベクタbytevectorのstartからendまでをUTF-8でデコードした結果の文字列を得ます。
start,endは省略されたときそれぞれ0,bytevectorの長さとなります。

Function: string->utf16 string
Function: string->utf16 string endianness
Function: string->utf16 string endianness endianness-mandatory

[R6RS]バイトベクタbytevectorをUTF-16でデコードした結果の文字列を得ます。
endianness-mandatoryが省略されているか#fのときは先頭のBOMでエンディアンを設定します。そうでないときはendiannessが使用されます。

Function: string->utf32 string
Function: string->utf32 string endianness
Function: string->utf32 string endianness endianness-mandatory

[R6RS]バイトベクタbytevectorをUTF-32でデコードした結果の文字列を得ます。
endianness-mandatoryが省略されているか#fのときは先頭のBOMでエンディアンを設定します。そうでないときはendiannessが使用されます。

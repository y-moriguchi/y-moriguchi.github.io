使用方法

ダウンロードしたjarファイルをクラスパスに追加して、net.morilib.lisp.Schemeのmainメソッドを実行してください。
$ java -jar ./schluessel.jar
 >(car '(1 2 3))
1
 >(exit)
$
現時点では、UTF-8で記述された日本語のコメントが存在します。
Eclipse等で編集するときは、文字コードをUTF-8にて指定してください。

SRFI-1, SRFI-13, SRFI-41は外部ライブラリです。これらのライブラリを使用するときは以下のようにしてインポートしてください。
 >(use srfi-1)
#<undef>

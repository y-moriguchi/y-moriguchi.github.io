morilib.net

morilib.net is a community which creates languages and tools.
Softwares created by morilib.net are as follows.

Schluessel: a Scheme interpreter A R5RS Scheme interpreter runs on Java 6 or above. Link
Morilib J: a J interpreter A subset of J interpreter runs on JavaScript. Link
Morilib APL: an APL interpreter An APL interpreter runs on JavaScript. Link
Arrycer: an array library An array library which can munipulate by axis like APL or NumPy. Link
Oiseau: a Programming Language based on lambda-calculus A programming language based on lambda-calculus. This is described by JavaScript. Link
epee:a calculator which treats exact pi and e A calculator which treats exact pi and e.
Calculating decimal and continued fraction expansion are supported. 		Link
